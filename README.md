# PM Prompt Library

A collection of prompts I use regularly in my product work.

Three categories:

- **[Discovery](./discovery/)** mock user interviews, assumption stress-testing, opportunity framing
- **[Data Interpretation](./data-interpretation/)** sense-checking metrics, challenging your own read of the numbers
- **[Content & AI Quality](./content-ai-quality/)** reviewing AI-generated content, classifying failure modes, rubric design

Each prompt includes context on when I use it and how to adapt it.
